== Description ==

Shows the RSS feeds of raspifeed.de on your wordpress blog.

== Installation ==

Activate the plugin from the **Plugins** menu in WordPress Dashboard.
To activate the Widget on the sidebar goto **Appearance** and select **Widgets**. 
Drag and drop the `Wel!RaspiFeed` widget to your sidebar.
